Installation

macOS

Using Homebrew:

brew tap kiwamizamurai/tap
brew install ghif

Or download the binary directly from GitHub Releases and add it to your PATH.

Quick Usage


# In your GitHub repository directory
ghif

# Specify repository
ghif --repository owner/repo

# Choose output format
ghif --format markdown  # default
ghif --format xml

# Fetch specific issues
ghif --issues 1,2,3
Note: A GitHub Personal Access Token is recommended for higher API rate limits. Set it in your environment: 
export GITHUB_TOKEN=$(gh auth token)
